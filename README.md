# AI Resume Analyzer

AI Resume Analyzer is a web application built using Python Flask.

## Features
- Upload PDF Resume
- Extract Resume Text
- Detect Skills Automatically
- Calculate Job Readiness Score
- Performance Analysis

## Technologies Used
- Python
- Flask
- HTML
- CSS
- PyPDF2
